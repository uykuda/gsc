# gscApi
![](logo)

#Setup Ngrok
`> gsc.exe ngrok <your authtoken>`
#Creating minecraft server
`> gsc.exe minecraft create <serverName> <serverSoftware> <serverVersion> <Port>`
#Load server
`> gsc.exe minecraft load <serverName> <Port>`
